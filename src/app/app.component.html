Duke Code Challenge: API

Thank you for participating in our code challenge!

For this challenge you will be using a mock api located here: https://jsonplaceholder.typicode.com/todos/

With this api, please do the following:

  1. When the user enters a number from 1-100 in the input below and clicks "fetch to-do", you should use the api service to send a request to https://jsonplaceholder.typicode.com/todos/[number]
  2. If a response is returned, you should display the to-do's title in the paragraph with the class "result".
  3. If an error is returned, or if the number is out of range, you should display the text "Sorry, the to-do was not found"
